# Student Depression Prediction Using Machine Learning
Overview: This project predicts student depression risk using machine learning based on lifestyle and academic factors. It includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and deployment using Streamlit.

# Dataset: 

Source: Kaggle – Student Depression & Lifestyle Dataset
Type: Binary Classification
Target: Depression (0 = Not Depressed, 1 = Depressed)

# Models Compared:

Logistic Regression
Decision Tree
Random Forest
K-Nearest Neighbors (KNN)
Support Vector Machine (SVM)

# Final Model: 
** Logistic Regression **
